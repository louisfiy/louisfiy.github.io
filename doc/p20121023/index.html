医学生的非恋爱日常

作者睡懵之下为友人写的文艺演出剧本。

第一场

【时间：午夜。

【地点：某聚会场所。

【人物：男A，女B，女C，男D，男E。

【布幕：某次聚会，A、B、C、D、E坐一桌，另可安排另一桌打酱油的。

众人：（起身）干杯！（一饮而尽后坐下）

C：诶，不如我们来玩真心话大冒险吧？

D：呵，这主意不错。

B：不要了吧，感觉好幼稚……

A：难得今天大家伙开心，一起玩吧。

E：是啊。

【呃……规则我不懂，😓，总之就是第一盘是B输。

C：哈哈，你输啦！真心话还是大冒险？

B：愿赌服输。（想了想）嗯，还是真心话吧……

C：那……

A：（抢问）你有没有男朋友？

B：（脱口）没有。

E：班长，你问这样的问题是不是对人家XX姑娘有意思啊？

A：别胡说，哪有的事……

【众人哗然。第二轮是A输。😤

C：（示意A做出选择）嗯哼？

A：（思考ing）那我就选……大冒险吧，嘿嘿！

D：（大嚷）我强烈建议让班长向XX（即B）表白！

A：……

E：叫你上你就上，扭扭捏捏不像样！

A：……

C：好啦好啦，既然班长大人不想大冒险，我们也就不要强求了；不过，你得自罚三杯！

D/E：对对对！

【A喝完三杯，众人再次举杯。

C：祝我们友谊地久天长！

众人：Cheers！


第二场

【时间：夜。

【地点：学校操场（或其它）。🥱

【人物：A，B。

【布幕：操场上，A和B静静地走着，没有说话。

A：其实……

B：嗯？

A：其实，我一直有些话想和你说。

B：嗯？

A：XX，我喜欢你！我是真的很喜欢你！我一直在寻找一个合适的机会，我也一直在权衡我这样做的利与弊，可是我觉得有些话还是说出来痛快，我也不想让这些事一直困扰着，毕竟如果你真的喜欢一个人，就不应该错过。（🥺，好像在哪见过这台词……）

B：对不起……我、我们还是做好朋友吧。

【晴天霹雳，天雷滚滚，六月飞霜，山崩地裂……


第三场

（🙄，这一场不想要的话可以删掉）

【时间：黄昏（或任意）。

【地点：医学院某解剖教室。

【人物：A，B。

【布幕：开场时，A正在做解剖实验（为达特殊效果可用西餐刀叉代替手术刀）；B在一旁看书，沉思。

B：（突然把书放下）我有些话想和你说。

A：（木有抬头继续做实验）啥？

B：我们分手吧！

A：（一惊）啥？！（抬头）

B：我觉得我们不合适。相处久了，我发现我们的共同话题、兴趣爱好越来越不一样，这样正常吗？这不科学……

A：科学？我不知道，我不在乎，但是难道我们在一起不开心吗？

B：你很开心？

A：你不开心？

B：也许我的心胸不够宽广，发生的一些事情总让我介怀；我觉得你……我并不是说你有很多缺点，只是你表现出来的可能正是我不感兴趣或不喜欢的……

A：XX，我们有些地方是意见不和，但是……我做错的我可以改啊！

B：谢谢你在这段时间带给我的温暖和快乐，谢谢你对我这么好，我们依然可以是好朋友……

A：够了，不要跟我来这套，想都别想，这算什么？小孩子过家家？

B：XX，我喜欢你，只是这不是我想要的感觉……我累了。（转身离去）

A：（愤然摔刀）Damn it!（木有追出去）


第四场

【时间：黄昏（或任意）。

【地点：医学院某解剖教室。

【人物：A，C，D，E。

【布幕：A坐在椅子上，C、D、E坐于A对面。开场时A在玩弄手中的刀具，心不在焉。

C：你的事我们都听说了，我们是来帮你的。

A：怎么帮？

C：首先，你把刀子放下。（A放下刀子）

D：（递给A一酒杯）把这喝了。（A接过酒杯）

E：那是什么？

D：伏特加。

E：搜噶～

A：嗝……

C：好吧，只管从头告诉我们发生什么事了。

A：我表白失败了…… // 我失恋了……

【冷场。

C：就这样？

A：就这样。

C：没了？ // 说散就散？

A：没了。 // 说散就散。

D：一点儿道理都没有……

C：说不定她当时心情不好。 // 说不定她只是心情不好。

D：内分泌失调？

C：经期症候群？

A：你哪里懂什么经期症候群了。

C：比你懂。

E：然后呢？

A：就这样……没了。 // 就这样……分了。

D：Jesus!

E：别想太多了……

C：就像大家所说的，天涯何处无芳草。

D：重点是，你会遇到更好的人，你会忘了她。

A：我忘不掉。

C：会的。你会忘掉的。（起身）Come on!（伸出手）

A：What?

C：Show一段吧，把不开心的事情通通忘掉。

A：（苦笑）

D：（问E）XX，此事你怎么看？

E：那就show一段呗。

【C拉着A来到教室外的空地上（之类的空旷平台什么的……），A发现班里的许多同学都来为他打气加油。众人齐喊：“班长不哭，班长加油！”场面极为壮观，音乐声起，happy ending！（解剖台上的小尸尸也可以参与到舞蹈的行列，💀）。


（正戏部分）


【舞蹈后若有多余的时间，可安排多一个结局。


【结局一　悲情版 😗

【时间任意；地点江边；人物BC。

C：你就这么狠心拒绝他了？

B：那还能怎样，他那么优秀，我不想拖累他……XX，答应我，永远不要让他知道我的病情，答应我，好么？

C:（不语）

B：XX……

C：（仍不语）

【B突然眼前一阵晕眩，接着而来的疼痛很快把她打垮在地。

C：XX，XX，快来人啊，XX……

【幕落，全剧终！


【结局二　温馨版 😊

【时间任意；地点江边；人物AC。

C：怎么样，心情还OK吧？

A：还好啦，死不了……（顿了一会儿）XX……

C：嗯？

A：谢谢你对我的鼓励！谢谢！

C：谢啥谢啊，兄弟有难我当然要帮啦。

A：（眼望前方，喃喃道）你真好……

【C一只手靠在江边的栏杆上，另一只手轻轻拂去垂在额头上的乱发，没有说话。许久，C拍了拍A的肩膀。

A：嗯？

C：走吧，去吃宵夜——你请客。

【幕落，全剧终！


号外！号外！喜大普奔，作者已经被友人捶死了。
