被偷走的一天

2014年1月31日
2.7千字

0

“《后天》是一部电影……”

小洛气呼呼地挂掉男朋友的电话，她实在不知道如何跟一个Geek谈感情——当小洛满心欢喜地问道：“亲爱的，你知道后天……”时，她得到了一个几乎让她抓狂的答案。

“气死我了！”小洛把手机往床上一摔，舍友小J从被窝中探出脑袋说：

“我的姑奶奶，谁又招你惹你了？”

“还能有谁……”小洛正想blabla一番，小J打断她的话：

“你就知足吧，咱宿舍就你一个人谈恋爱了，而且泡的还是物理系的系草，你还有啥好抱怨！”

“……”

——那是前天的事儿了。

今天又是雨天，灰蒙蒙的，凉飕飕的，仿佛一时半会停不了。小洛很不喜欢，下雨意味着很多计划会被打乱。

计划，好像也是前天的事情。

小洛的计划是这样的：首先，不出意外，昨晚她就能拥着男友甜蜜的祝福入眠；然后今天起床洗漱完毕化个美美的妆，接着就是吃饭逛街购物看电影玩转一整天的节奏，咩哈哈……

嗯，这就是计划。简单整洁的计划。

如果，一切都那么完美就好了。

今早醒来已是清晨，爪机还握在手里，小洛睡眼朦胧，迷迷糊糊看了一下——未读短信：0。

“好吧，果然是想太多了。他那么忙，怎么会记得……”小洛安慰自己，一翻身，一觉睡到午后。

午后，阳光正隐。

小洛坐在靠近窗台的书桌前，情绪有些忧郁，双手托着脸颊，唉声叹气，打不起精神，就像来了大姨妈。

正在一旁听歌的小J摘下耳机，惊讶地望着她：“我的姑奶奶，您今天是肿么啦？”

“他不接我电话……从前天到现在。就回了个信息说在做实验……”小洛抱怨着。

J说：“我看你是纯粹自虐吧！”她给出了自己的想法。没等小洛回答，她已经戴上耳机听歌了。

话已至此，小洛开始伤感：“好不容易难得今天生日，我只是希望他能好好陪我，这有错么？”

还在一旁听歌的小J再次摘下耳机，她因惊讶而张大的嘴巴足以塞入一个苹果，那异常奇怪的目光如机关枪般对着小洛疯狂扫射，诧异的声音从喉咙深处喷射而出：“你说什么啊你？你的生日不是在昨天已经过了吗？”

古往今来，时间就像离弦的箭，一去匆匆，覆水难收。随着生产力的发展，心有不甘的青年们一旦吃饱喝足，就会不约而同地幻想着“时间旅行”。有的人用它写小说拍电影，有的人靠它做实验发论文——前者名利双收，而后者的研究结果则更具合理性。但是奇怪的是，当科学尝试向人们解释“时间是一维”时，许多人表示不能完全理解！理论派对此无能为力，只好求助实验派；实验派仰望星空，思前想后，终有所获。

套用一句话，因为问题就在那里，所以总得解决。

我们学校向来以理工科见长，到了我们这一届更是人丁兴旺，其中不乏喜欢研究时空的顶级天才。“时即是空”是学校历史最悠久的社团，是实践派崭露拳脚的地方，他们通过售卖诸如对撞机之类的技术搞到一笔钱，用于制造类似时间机器的可怕设备。据我所知，这些设备相貌丑陋、味道恶心，直教人退避三舍。

我最终卷入这场风波完全是因为苏小。

苏小是我的“青梅竹马”，也是小洛的男朋友，更是“时即是空”的联合创始人。他找到我的时候，我正蹲在茅坑上努力。

“陈老师，我看你来了！”他毫无防备地闯入宿舍，顿时令我便意全无。

“你丫别那么冲动好伐？没看到我正忙着吗？”我隔着门对着他嚷，“有话快说，谁不知道您老是大忙人，来看我？笑话！”

他也不扭捏，直入正题：“我们协会新出了个新鲜玩意儿，想邀请你体验一把，名额有限，欲试从速。”

我立马变得警觉，无事献殷勤非奸即盗：“你会这么有心？”

“你可以不相信我，但是你不可以不相信我的机器。倘若你加入这次的试验，这绝对会成为你毕生难忘的一次经历！”

“机器？”

“是滴！‘Time Machine’！”他激动地说，“我们发现，与其空谈时间旅行可能导致的种种悖论，不如亲身测试一番。”

“扯淡吧你！物理学第一定律告诉我们：时间不可逆！”

“恰恰相反！”苏小毫不介意我的无知，耐心地解释，“在物理学的规律中，时间这个变量不但可以向前，还可以向后，之所以看起来时间连续不变一直向前，是因为我们无法观测到时间的离散变化。我们知道，时间在我们所处的这个宇宙中的流逝体现在‘熵值’的渐进性积累叠加。根据热力学定律，热量无法自发地从温度低的物体传递到温度高的物体，在这个不可逆的热力过程中，熵的微增量恒大于零，因此宇宙永远处于熵增的状态。打个比方，假如我们要扭转时间的箭头，就意味着我们需要让宇宙熵减。虽然，系统的演化总是从有序趋于无序，并且这个过程是不可逆的，但是系统内部单个分子的运动却是可逆的，也就是说，热力学定律并不适用于微观系统。19世纪英国物理学家麦克斯韦曾经设想过一个‘智能’的‘麦克斯韦妖’，以实现一个封闭体系的熵减过程！……”

“得了吧你！”我毫不客气地打断苏小慷慨激昂的发言，“要让‘麦克斯韦妖’完成复杂的‘识别工作’，不可避免地会引入第二个能量传递过程，而且第一个过程中熵值的减少并不能抵消第二个过程中熵值的增加，所以在整个过程中，熵值实际上还是增加了，正如你所说的：‘宇宙永远处于熵增的状态’，所以时间还是无法逆转！”

“你说得对！”苏小并没有反驳我的观点，继续说道，“在量子世界中，道理也是类似的，不同的是，这里的‘熵’指的不是混乱度而是信息；从这个角度讲，宇宙是一个信息不断增加的系统，要想让时间倒流，就必须让宇宙的信息减少。问题就出在这儿，即便我们能够经历一个熵减的过程，但是相应的，曾经有过熵增的记录也会消失。如果一个事物根本就没有在宇宙中留下任何痕迹，那么只能说，这个事物根本就没有在宇宙出现过。所以，即使宇宙中曾经出现过时间逆转的事件，由于没有任何可以观察到的记录，我们也无从知晓时间是否倒流过。”

“那你说个吧唧啊！”我开始怒了，“既然理论上行不通，那么你们所谓的时间机器意义何在？”

“我原谅你的物理学只有高中水平！”苏小显然信心十足，实验派的气概毕露无遗，“不过，从技术上讲，我们的确无法进行物质层面上的时间旅行。事实上，我们的机器只是利用时空中的‘小间隙’，制造出一个时间上独立于我们这个宇宙的封闭空间，如果把这个空间的‘熵变’映射到这个空间的时间上，那么技术上，我们是可以随意拉伸、压缩这根时间轴的。”

“怎么做？”

“反过来，在不改变时间的情况下，改变单位时间内一个系统中熵变的量。举个例子，熵增在人体最直接的体现就是记忆的增加。懂了没？——虽然我们所做的实验并不是传统意义上的时间旅行，但是这对于时空的研究，我相信我们已经迈出了一大步！”

“既然你们万事俱备，为何还要把我这个门外汉扯进来？哥高中学的虽然是理科，可大学读的是文学啊亲！”

“喔哦噢！恐怕是人人为我我为人人。你看，我和我的小伙伴们这么厉害，别人知道吗？所以，博学多才学富五车的你，很荣幸能为我们记下这一时刻。”

“真的？”

“非假！”

“非假？”

“真的！”

咕咚！——我猜我该冲马桶了。
