🚀 写在前面

从零开始的静态博客网站，基于 Astro 搭建。样式使用 Tailwind CSS + Stylus 编写，风格受 Astro Blog 和 AstroPaper 模板启发——讲真本人没啥设计天赋就是了😅。（搬迁工作乱序进行中……）

上一版网站用的是 Hexo 框架 + Fluid 主题，有图为证：

好处是几乎无需编码也能 build 一个漂亮的网站，缺点是受制于主题。不过现在打算推翻重做了。虽然如此，你仍然可以从 Hexo 的“开发日志”中获得灵感。
